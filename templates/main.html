Recruitment task

Go to dq-dataset.herokuapp.com/data to retrieve an example dataset.

Your goal is to prepare a process/visualisation catching data quality issues in the above dataset.

The data table contains the following information:

  • id: integer unique for each row, must not be empty.
    Example values: 0, 1, 2
  • date: date in ISO format (YYYY-MM-DD).
    Example values: 2018-06-01
  • view_count: integer number describing the number of occurrences of an arbitrary event.
    Example values: 1, 10, 123
  • view_count_by_category: view_count split by categories saved as a string representation of a JSON object.
    Example values: {"category_0": 5, "category_1": 13}

The goal is to develop and automate test cases to discover data quality issues.

Find the rows which values do not comply with the assumptions.

Present the results in any way you like. Remember to attach any code you used to solve the problem.
